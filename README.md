T# 🌸 VAIDEHI OS

A personal student operating system built with **Next.js** and **Tailwind CSS**.

VAIDEHI OS is a personal productivity dashboard designed for students with study tracking, cybersecurity learning, and planning tools.

---

## ✨ Features

- 📚 Semester Dashboard
- 📝 Subject Hub & Resources
- 💻 Cyber Security HQ
- 🎓 NPTEL Learning Space
- 📅 Calendar Planner
- 💼 Career Tracker
- 🌸 Personal Productivity Section

---

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide Icons
